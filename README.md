# Mi Serie para Aproximar π/2

## La fórmula que descubrí:

\[
\frac{\pi}{2} \approx 6\sum_{n=1}^{\infty} (-1)^{n+1}\left(\frac{2}{7}\right)^n\left(\frac{8}{25}\right)^{\,n-1}
\]

## Resultado:
- **Valor aproximado**: 1.57068
- **Error**: 0.00737%
- **Forma exacta**: \(\frac{300}{191}\)
